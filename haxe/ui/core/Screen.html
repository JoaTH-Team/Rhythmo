References the main application's screen, in a cross-framework way.

Constructor

new()

Creates a new Screen.

Usually, you wouldn't want to create a screen yourself, but to use the one in Screen.instance. Double check if thats what your'e trying to do.

Variables

@:value(null)currentMouseX:Null<Float> = null

The x position of the mouse on screen.

A lower value means the mouse is at the left side of the screen, whie a higher value means the mouse is at the right side of the screen.

@:value(null)currentMouseY:Null<Float> = null

The y position of the mouse on screen.

A lower value means the mouse is closer to the top of the screen, whie a higher value means the mouse is closer to the bottom of the screen.
