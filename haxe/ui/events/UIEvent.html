Whether event will be dispatched to parent component

Whether the event was canceled. If canceled, it won't be dispatched to parent. Some events effects can be avoided if canceled

Usually data that accompanies a property changed

previousValue:Variant

Sometimes, you can access the previous value of the component in a CHANGE event

An event might have relatedComponent which is different to the target. For example, ADDED_COMPONENT will have the new child component as a related component
