typedef DragOptions

package haxe.ui.dragdrop

Fields

@:optionaloptionalmouseTarget:Null<Component>

A component that will trigger dragging - usually the draggable component or a sub-component of it. Default: the draggable component

@:optionaloptionaldraggingStyleName:Null<String>

A style name to add to draggable components while being dragged Default: dragging

@:optionaloptionaldraggableStyleName:Null<String>

A style name to add to draggable components Default: draggable

@:optionaloptionaldragTolerance:Null<Int>

The distance the mouse must travel while mouseDown on the mouseTarget before drag begins. Default: 1

@:optionaloptionaldragOffsetY:Null<Float>

The y offset to be added to the moveTarget during drag in addition to the mouse y offset from the moveTarget's y. Default: 0

@:optionaloptionaldragOffsetX:Null<Float>

The x offset to be added to the moveTarget during drag in addition to the mouse x offset from the moveTarget's x.. Default: 0

@:optionaloptionaldragBounds:Null<Rectangle>

A Rect specifying the bounds in the screen's coordinate space. Default: screen's bounds
